1022 用getline读取一行的用法是getline(cin,s)，另外要注意如果之前有cin，直接getline会读换行符；当读取一行用空格分开的string时，可以用istringstream来分离出每一个string
```C++
getline(cin,k);
istringstream kk(k);
string kw;
vector<string>gr;
while(kk>>kw){
    gr.push_back(kw);
}
```
